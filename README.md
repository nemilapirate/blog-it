# Projet type blog avec les technologies React et TypeScript

Ce projet à été fait avec Vite.
Pour cloner : 
    - git clone + nom du projet

Pour le lancer en local :
    Pour les dépendances : 
        - npm i
    Ensuite :
        - npm run dev
